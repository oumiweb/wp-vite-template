【開発環境セットアップ手順】

この静的開発サーバー（http://localhost:5173）は起動したままでOKです。

WordPressテーマ開発を行う場合は、別ターミナルで以下を実行してください：

  1. Docker Desktop を起動（ステータスバーに "running" と表示されていることを確認）
  2. WordPress環境を起動
    $ yarn wp-start
    → 初回は5-10分程度かかります
    → 完了すると http://localhost:8888 でWordPressにアクセスできます
  3. WordPress初期設定を適用（初回のみ）
    $ yarn wp-init
    → タイムゾーン・パーマリンク・フロントページ・シード投稿を自動設定
    → テーマの有効化も自動で実行されます（手動操作不要）
  4. WordPress管理画面にログイン（初回のみ）
    → URL: http://localhost:8888/wp-admin/
    → ID: admin / パスワード: password
    → 言語設定は既に日本語になっています（.wp-env.json の設定により自動適用）
  5. 開発作業
    → src/ フォルダ内でHTML/SCSS/JavaScriptを編集
    → 変更は自動的に反映されます（手動リロードが必要な場合あり）
    → WordPress環境（localhost:8888）で確認しながら開発

終了時: $ yarn wp-stop

詳細は README.md を参照してください。
